# 予備実験の振り返り

ResNetの学習済みモデルを使ったら案外簡単に学習できたが、課題も多い。

## Keep(良かったこと)

- ResNetの学習済みモデルを使っていく

## Problem(問題)

- ImageNetという綺麗な画像で学習されたモデルなので、ノイズが酷い画像の検出精度を確認すべき
- 物体が一つだけ映った画像で学習しているので、複数の物体が映った画像で一つでもお題の物体が写っていることを検出できるプログラムが必要
- クラス分類が細かすぎるので、おおまかな分類（バス、車、消火栓、信号）のようなざっくりとしたクラスに再設計する必要がある
- 一番自信がある答えしか見ていないので、確率が低い答えが正解の場合もある。

## Try(次やること)

- クラス分類をよりおおまかなものにする
- 複数の物体が写ったデータセットを用意する
- ノイズが含まれた画像をデータセットに追加する
- データセットの数を増やす
- 画像の煩雑さを増やす(複数の物体を含めた画像を追加する)
