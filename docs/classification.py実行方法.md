# classification.py 実行方法

## このスクリプトは何をするもの？

reCAPTCHA の「バスが写っている画像をすべて選んでください」のような問題を、AI に解かせる
プログラムです。フォルダの中にある画像を1枚ずつ見て、「お題（例: バス）が写っているか」を
判断し、写っている画像の番号を出力します。

裏側では **ResNet18** という、すでに学習済みの画像分類 AI を使っています。
自分で AI を学習させる必要はなく、画像を渡すだけで使えます。

---

## まず動かしてみる

ターミナルで次のコマンドを実行します。

```bash
uv run python src/classification.py --image-dir img --target bus --threshold 0.05
```

- `uv run` … 必要なライブラリ込みで Python を実行するコマンド（環境構築済みなら気にしなくてOK）
- それ以降が、このスクリプトへの「指示」です（下の表で説明します）

---

## 指示（引数）の意味

| 引数 | 何を指定するか | 例 | 省略時 |
|---|---|---|---|
| `--image-dir` | 調べたい画像が入っているフォルダ | `img` | `img_bus_rain` |
| `--target` | 探したいお題 | `bus`（日本語の `バス` も可） | `bus` |
| `--threshold` | 「お題あり」と判断する厳しさ（後述） | `0.05` | `0.05` |

---

## 判断のしくみ（3ステップ）

### ステップ1：画像を AI に見せる

各画像を ResNet18 に入力すると、「この画像は何か」を 1000 種類の候補について
**確率**で答えてくれます（合計するとちょうど 1.0 になります）。

> 例: ある画像 → `school bus: 0.7`, `minibus: 0.2`, `dog: 0.01`, ...

この確率は **softmax** という計算で 0〜1 に整えられた「自信の度合い」だと思ってください。

### ステップ2：お題ごとに確率をまとめる

ResNet18 の答えは細かすぎる（`school bus` と `minibus` が別物扱い）ので、
お題の `bus` に関係するクラスをまとめて1つの**スコア**にします。

> `school bus(0.7) + minibus(0.2) = バスである確信度スコア 0.9`

このスコアが「この画像にバスが写っている自信」を表します（0〜1、大きいほど自信あり）。

### ステップ3：閾値（しきい値）で線引きする

スコアが `--threshold` 以上なら「お題あり」として選びます。

`--threshold` は**線引きの厳しさ**です。

- **下げる**（例 0.01）→ 自信が低くても選ぶ → **取りこぼしは減るが、間違いも増える**
- **上げる**（例 0.5）→ 自信が高いものだけ選ぶ → **間違いは減るが、取りこぼしが増える**

> 「取りこぼしの少なさ」を再現率、「間違いの少なさ」を適合率と呼びます。
> この2つはトレードオフ（片方を良くするともう片方が悪くなる）の関係にあります。

---

## 出力の読み方

### ① 各画像のスコア

画像ごとに、お題の確信度スコアが表示されます。

```text
スコア [img/1.jpg]: bus=0.953   ← バスの自信が高い
スコア [img/2.jpg]: bus=0.000   ← バスではなさそう
```

### ② 読み込んだ画像一覧

フォルダ内の画像を自動ですべて読み込みます（番号順に並びます）。

```text
読み込んだ画像: ['img/1.jpg', 'img/2.jpg', 'img/3.jpg', 'img/4.jpg']
```

`img/10.jpg` のようなファイルも自動で対象になるので、手で並べる必要はありません。

### ③ 最終的な選択結果

「お題あり」と判断された画像の**番号**が出ます。番号は **0 から**数えます。

```text
最終的な選択結果: [0, 2, 3, 4, 5, 6, 7, 26, 31, 57]
```

`0` は読み込んだ画像一覧の**1枚目**（`img/1.jpg`）を指します。

---

## 補足

- どの `--threshold` が良いかは画像の種類（きれいな画像 / 雨でノイズの多い画像）で変わります。
- `predict_image_labels()` という関数は「AI が何だと思ったか」を確認するための補助で、
  選択の判断そのものには使っていません。
- このスクリプトは単体で動きます（追加の特別なライブラリは不要）。
